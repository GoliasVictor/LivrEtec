Buscar(ParamBuscaEmprestimo)

Busca todos os usuarios que satifazem a seguintes condições todas as seguintes condições

  • Caso parametros.IdLivro não seja null apenas os emprestimos do livro indicado por parametros.IdLivro
  • Caso parametros.IdPessoa não seja null retornara emprestimos que foram emprestados pela pessoa indicada por parametros.IdPessoa
  • Caso parametros.Fechado seja true retornara apenas os emprestimos fechados
  • Caso parametros.Fechado seja false retornara apenas os emprestimos abertos
  • Caso parametros.Atrasado seja true retornara apenas os emprestimos atrasados
  • Caso parametros.Atrasado seja false retornara apenas os emprestimos que não estão estão atrasados

Parameters

ParamBuscaEmprestimo parametros
Returns
Type Description
Task<IEnumerable<Emprestimo>>

Emprestimos apos todos
Examples

Esse codigo busca todos os emprestimos abertos que estão atrasados

await emprestimoService.Buscar(new ParamBuscaEmprestimo(
    IdLivro: null,
    IdPessoa: null,
    Fechado: false,
    Atrasado: true
))

Já este busca todos os emprestimos já fechados do livro de id 5

await emprestimoService.Buscar(new ParamBuscaEmprestimo(
    IdLivro: 5,
    IdPessoa: null,
    Fechado: true,
    Atrasado: null
))
